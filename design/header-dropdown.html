Header Dropdown Demo

Click the "Events" link in the navigation to see the dropdown menu with glass effect, rotating chevron, and staggered slide-in animation.

Design Notes

Glass Effect

Uses backdrop-filter: blur(24px) with a semi-transparent background rgba(10, 10, 10, 0.85) for the frosted glass appearance. A subtle bottom border adds definition.

Chevron Animation

The chevron rotates 180¬∞ using transform: rotate(180deg) when the menu is expanded. Uses a smooth cubic-bezier timing function for natural movement.

Slide-Down Animation

The panel transitions from translateY(-8px) and opacity: 0 to visible state. Menu items have staggered delays (50ms increments) for a cascading reveal effect.

Accessibility

Uses proper ARIA attributes: aria-expanded, aria-controls, and role="menu". Closes on Escape key and clicking outside.

Content Structure

Events are organized into two columns: Upcoming and Past. Each item shows date, title, and venue/winner. Past events display the winning band with a trophy emoji.
